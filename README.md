# learnspace

Личная база знаний по Computer Science: структурированные конспекты от фундамента до продвинутых тем. Цель — глубокое, детальное понимание, достаточное для сложных алгоритмических собеседований и системного программирования.

## Структура

```text
learnspace/
├── computer/                 # Как устроен компьютер
│   ├── architecture/         #   архитектура ЭВМ
│   ├── networks/             #   компьютерные сети
│   └── os/                   #   операционные системы (в т.ч. linux/)
└── programming/              # Программирование
    ├── algorithms-and-data-structures/   # Алгоритмы и структуры данных (16 частей)
    ├── dbms/                 #   базы данных (sql/)
    ├── languages/            #   языки (c/, cplus/)
    └── vcs/                  #   системы контроля версий (git/)
```

## Алгоритмы и структуры данных

Главный раздел — [`programming/algorithms-and-data-structures`](programming/algorithms-and-data-structures): 16 частей (папки `part-NN-*`) и десятки глав (файлы `NN-*.md`) в kebab-case. Порядок частей = рекомендуемая траектория изучения.

| Часть | Тема |
|---|---|
| 00 | Фундамент: модель вычислений, анализ сложности, математика |
| 01 | Базовые структуры данных |
| 02 | Сортировки и порядок |
| 03 | Деревья |
| 04 | Графы |
| 05 | Парадигмы проектирования (greedy, D&C, DP, рандомизация) |
| 06 | Поиск и техники на массивах |
| 07 | Запросы на диапазонах (RMQ, segment tree, декомпозиции) |
| 08 | Системы непересекающихся множеств (DSU) |
| 09 | Строковые алгоритмы |
| 10 | Математика для алгоритмов |
| 11 | Вычислительная геометрия |
| 12 | Продвинутые структуры данных |
| 13 | Сложность, пределы, приближения |
| 14 | Параллельные и системные аспекты (C++) |
| 15 | Мастерство решения задач |

## Формат конспектов

Каждая глава строится по принципу: **идея → интуиция → доказательство (инварианты/сложность) → production-grade код на C/C++ → задачи**. Сложность по времени и памяти указывается всегда; разбираются крайние случаи и UB.
